## 📏 TRADING RULES (NEVER BREAK THESE)

### Position Sizing Formula

```text
Shares = (Account × Risk%) / (Entry - Stop)
Example: ($5000 × 1%) / ($50 - $49) = 50 shares
```

### The Unbreakable Rules

1. **Risk Management**
   - Never risk more than 1-2% per trade
   - Stop loss on EVERY trade (no exceptions)
   - Maximum 5% daily loss (STOP if hit)
   - Maximum 10% weekly loss (Paper trade if hit)

2. **Entry Discipline**
   - Only A+ setups (all criteria met)
   - Risk/Reward minimum 2:1
   - No chasing or FOMO entries
   - Plan before market opens

3. **Psychological Rules**
   - No revenge trading (EVER)
   - After 2 losses = done for day
   - Check emotional state before trading
   - Journal honestly (especially losses)

4. **Process Focus**
   - Right process > Right outcome
   - Small consistent gains > Home runs
   - Education daily (minimum 1 hour)
   - Review every trade

## 🚨 EMERGENCY PROTOCOLS

### Financial Emergencies

- **Daily Loss Limit Hit (-5%)**: Stop immediately, review tomorrow
- **Weekly Loss Limit Hit (-10%)**: Paper trade only for 1 week
- **Account Drawdown >20%**: Full stop, complete audit required

### Technical Emergencies

- **Platform Crash**: Schwab: 1-800-435-4000
- **Internet Failure**: Mobile hotspot backup ready
- **Power Outage**: Laptop charged, mobile app ready

### Psychological Emergencies

- **Tilt/Emotional**: Close all positions, walk away
- **Revenge Trading Urge**: Shut down platform
- **Overconfidence**: Reduce position size 50%

### Life Emergencies

- **Family Crisis**: Trading stops, family first
- **Health Issues**: No trading when sick/tired
- **Job Loss**: Do NOT increase risk
